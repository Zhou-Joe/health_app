# 图片资源说明

## 需要准备的图片列表

由于这是健康管理系统，建议使用简洁的医疗/健康主题图标。

### 方案1：使用纯色占位图（快速测试）

可以直接创建简单的纯色图片，尺寸参考：

**Tab栏图标**（81x81px，建议png格式）
- home.png - 蓝色"首页"图标
- home-active.png - 深蓝色"首页"选中图标
- report.png - 绿色"报告"图标
- report-active.png - 深绿色"报告"选中图标
- upload.png - 紫色"上传"图标
- upload-active.png - 深紫色"上传"选中图标
- ai.png - 青色"AI"图标
- ai-active.png - 深青色"AI"选中图标
- me.png - 灰色"我的"图标
- me-active.png - 深灰色"我的"选中图标

**页面图片**
- logo.png（160x160px）- 蓝色背景+白色"+"号
- avatar.png（120x120px）- 灰色圆形头像
- empty.png（200x200px）- 浅灰色空状态图

### 方案2：使用在线图标库

推荐网站：
1. **iconfont** (https://www.iconfont.cn/) - 阿里巴巴图标库
2. **IconPark** (https://iconpark.oceanengine.com/) - 字节跳动图标库
3. **Flaticon** (https://www.flaticon.com/) - 国外图标库

搜索关键词：
- "医疗"
- "健康"
- "文档"
- "AI"
- "用户"

### 方案3：使用Emoji或简单图形（最简单）

可以暂时使用文字或emoji代替：
- Tab栏可以用纯色块+文字
- 或者直接用emoji（🏠、📄、⬆️、🤖、👤）

### 临时解决方案

如果不想准备图片，可以修改 `app.json`：

```json
"tabBar": {
  "color": "#666666",
  "selectedColor": "#4A90E2",
  "backgroundColor": "#FFFFFF",
  "borderStyle": "white",
  "list": [
    {
      "pagePath": "pages/dashboard/dashboard",
      "text": "首页"
      // 暂时删除iconPath，使用纯文字Tab
    },
    ...
  ]
}
```

### 图片尺寸要求

| 类型 | 建议尺寸 | 格式 |
|------|---------|------|
| Tab栏图标 | 81x81px | PNG |
| Logo | 160x160px | PNG/JPG |
| 头像 | 120x120px | PNG/JPG |
| 空状态图 | 200x200px | PNG |
| 上传图标 | 120x120px | PNG |

### 快速创建方法

使用在线工具：
1. **Canva** (https://www.canva.com/) - 在线设计工具
2. **Figma** - 专业设计工具
3. **PPT** - 最简单，导出为PNG即可

或者使用代码生成（推荐）：
- 搜索"在线图标生成工具"
- 选择颜色、添加文字
- 下载PNG格式
