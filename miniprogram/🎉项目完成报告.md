## 🔧 技术实现

### API对接
**基础路径**: `/api/miniprogram/`

**认证方式**: `Authorization: Bearer {token}`

**主要接口**:
- POST `/login/` - 用户登录
- GET `/user-info/` - 获取用户信息
- POST `/upload/` - 上传报告
- GET `/checkups/` - 获取报告列表
- GET `/indicators/` - 获取指标列表
- POST `/advice/` - 获取AI建议
- GET `/conversations/` - 获取对话列表
- POST `/integrate-data/` - 数据整合
- 等等...共20个接口

### 核心技术点
1. **HTTP请求封装** - 统一的请求拦截和错误处理
2. **Token管理** - 自动添加认证头
3. **文件上传** - 支持大文件上传
4. **实时进度** - 轮询处理状态
5. **分页加载** - 上拉加载更多
6. **错误处理** - 友好的错误提示
7. **状态管理** - 全局数据和页面数据

## 🚀 使用指南

### 第一步：配置服务器地址

编辑 `miniprogram/app.js` 第5行：
```javascript
baseUrl: 'http://your-server:8000'
```

### 第二步：配置AppID

编辑 `miniprogram/project.config.json` 第18行：
```json
"appid": "wx1234567890abcdef"
```

### 第三步：启动Django后端

```bash
cd /mnt/c/Users/ZC/VSProject/health
python manage.py runserver 0.0.0.0:8000
```

### 第四步：启动小程序

1. 打开微信开发者工具
2. 导入项目：选择 `miniprogram` 目录
3. 点击"编译"
4. 开始使用！
